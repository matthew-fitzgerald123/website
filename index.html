About Me

I'm an ML Infrastructure Engineer focused on building the platform-layer systems that make machine learning workflows production-ready and scalable. At Cognitive Network Solutions, I built and maintained infrastructure for model deployment and serving — including Kubernetes-based workloads, MLflow experiment tracking and model registry, and Kafka-driven data pipelines that fed ML systems in real time.

I worked across GCP, Azure, and AWS managing container orchestration, CI/CD pipelines for model artifacts, Terraform-provisioned infrastructure, and production observability with Prometheus and Grafana. The systems I built let data scientists ship models into production without managing deployment complexity themselves.

Previously, I interned at Dfinitiv where I built AWS-native backend services in Python to automate media curation workflows and reduce manual processing by over 60%.

I'm drawn to the infrastructure and platform engineering layer of ML — the reliability, scalability, and operational complexity that turns experimental models into dependable production systems.
